ХУРААНГУЙ

МОНГОЛ

ArenaHub нь AI болон gamification технологид суурилсан full-stack coding learning platform юм. Энэхүү систем нь хэрэглэгчдэд програмчлалын хичээл судлах, coding даалгавар бодох, AI assistant-аас тусламж авах, achievement болон leaderboard системээр дамжуулан идэвхээ нэмэгдүүлэх боломжийг олгодог.

Төслийн frontend хэсэгт Next.js 15, React 19, Tailwind CSS ашигласан бол backend хэсэгт Next.js API Routes, JWT authentication, Prisma ORM, PostgreSQL database ашигласан. Мөн Groq API дээр суурилсан AI tutor системийг хэрэгжүүлсэн. Систем нь нийт 23 database table, 30+ API endpoint, 280+ coding task бүхий production-level architecture ашиглан бүтээгдсэн.

Түлхүүр үг: gamification, AI tutor, coding learning platform, Next.js, PostgreSQL, JWT authentication

ENGLISH

ArenaHub is a full-stack coding learning platform built on AI and gamification technologies. The system enables users to study programming courses, solve coding tasks, receive assistance from an AI assistant, and increase their engagement through achievement and leaderboard systems.

The frontend was built with Next.js 15, React 19, and Tailwind CSS, while the backend uses Next.js API Routes, JWT authentication, Prisma ORM, and PostgreSQL. An AI tutor system based on the Groq API was also implemented. The system was built using a production-level architecture with 23 database tables, 30+ API endpoints, and 280+ coding tasks.

Keywords: gamification, AI tutor, coding learning platform, Next.js, PostgreSQL, JWT authentication

1. ОРШИЛ

Сүүлийн жилүүдэд online learning platform-ууд хурдацтай хөгжиж байгаа боловч хэрэглэгчийн идэвх, personalized learning experience хангалтгүй хэвээр байна. Ихэнх coding platform-ууд хэрэглэгчийг удаан хугацаанд идэвхтэй байлгах gamification system болон AI assistance дутмаг байдаг.

Иймээс ArenaHub платформыг боловсруулсан бөгөөд энэхүү систем нь дараах боломжуудыг нэгтгэсэн платформ юм:

  • AI-powered tutor — Groq API дээр суурилсан personalized assistant
  • Interactive coding environment — Monaco Editor ашигласан VS Code-style editor
  • Gamification system — XP, level, combo, HP, streak mechanics
  • Achievement & leaderboard — Global ranking болон reward system
  • Contest system — Мөнгөн шагналтай coding тэмцээн
  • Multi-language support — Монгол болон Англи хэлний дэмжлэг

Төслийн Зорилго

Програмчлал сурах процессыг тоглоом шиг сонирхолтой, interactive, personalized болгох замаар хэрэглэгчийн суралцах идэвх, мэдлэгийн чанарыг нэмэгдүүлэх.

ОНЦЛОГ

ArenaHub нь Монголд анх удаа gamification болон AI туторыг нэгтгэсэн coding learning platform юм. Систем нь Vercel болон Neon.tech дээр production-д deployed байна.

Validation & Error Handling

Бүх API request дээр Zod schema validation ашигладаг. 5xx алдааг client-д нуух замаар мэдээлэл алдагдахаас хамгаалдаг:

// handleError — 5xx errors are hidden from client
if (status >= 500) {
  console.error('[API Error]', e)
  return err('Серверийн алдаа гарлаа', 500)
}
