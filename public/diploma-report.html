Хураангуй

Монгол

ArenaHub нь AI болон gamification технологид суурилсан full-stack coding learning platform юм. Энэхүү систем нь хэрэглэгчдэд програмчлалын хичээл судлах, coding даалгавар бодох, AI assistant-аас тусламж авах, achievement болон leaderboard системээр дамжуулан идэвхээ нэмэгдүүлэх боломжийг олгодог.

Төслийн frontend хэсэгт Next.js 15, React 19, Tailwind CSS ашигласан бол backend хэсэгт Next.js API Routes, JWT authentication, Prisma ORM, PostgreSQL database ашигласан. Мөн Groq API дээр суурилсан AI tutor системийг хэрэгжүүлсэн.

Түлхүүр үг: gamification, AI tutor, coding learning platform, Next.js, PostgreSQL, JWT

English

ArenaHub is a full-stack coding learning platform built on AI and gamification technologies. The system enables users to study programming courses, solve coding tasks, receive assistance from an AI assistant, and increase their engagement through achievement and leaderboard systems.

The frontend was built with Next.js 15, React 19, and Tailwind CSS, while the backend uses Next.js API Routes, JWT authentication, Prisma ORM, and PostgreSQL. An AI tutor system based on the Groq API was also implemented.

Keywords: gamification, AI tutor, coding learning platform, Next.js, PostgreSQL, JWT

1. Оршил

Сүүлийн жилүүдэд online learning platform-ууд хурдацтай хөгжиж байгаа боловч хэрэглэгчийн идэвх, personalized learning experience хангалтгүй хэвээр байна. Ихэнх coding platform-ууд хэрэглэгчийг удаан хугацаанд идэвхтэй байлгах gamification system болон AI assistance дутмаг байдаг.

Иймээс ArenaHub платформыг боловсруулсан бөгөөд энэхүү систем нь дараах боломжуудыг нэгтгэсэн:

  • AI-powered tutor — Groq API дээр суурилсан personalized assistant
  • Interactive coding environment — Monaco Editor ашигласан VS Code-style editor
  • Gamification system — XP, level, combo, HP, streak mechanics
  • Achievement & leaderboard — Global ranking болон reward system
  • Contest system — Мөнгөн шагналтай coding тэмцээн
  • Multi-language support — Монгол болон Англи хэлний дэмжлэг

Төслийн Зорилго

Програмчлал сурах процессыг тоглоом шиг сонирхолтой, interactive, personalized болгох замаар хэрэглэгчийн суралцах идэвх, мэдлэгийн чанарыг нэмэгдүүлэх.

Онцлог

ArenaHub нь Монголд анх удаа gamification болон AI туторыг нэгтгэсэн coding learning platform юм. Систем нь Vercel болон Neon.tech дээр production-д deployed байна: btbn-arenahub.vercel.app

7. Frontend Хөгжүүлэлт

Frontend хэсгийг Next.js 15 App Router architecture ашиглан боловсруулсан. Pixel-art dark cyberpunk aesthetic бүхий UI дизайн хэрэгжүүлсэн.

Гол Онцлогууд

  • Responsive design (Mobile + Desktop)
  • Monaco Editor — VS Code style coding
  • Three.js 3D pixel character rendering
  • 21 улсын SVG pixel art flags
  • Game HUD — HP, XP, combo display

State Management

  • AuthContext — User session, login/logout
  • LanguageContext — МН/EN switching
  • useState/useEffect — Local state
  • useSyncExternalStore — SSR-safe hydration

8. Backend Хөгжүүлэлт

Backend нь Next.js API Routes дээр суурилсан serverless architecture ашиглан бүтээгдсэн. Нийт 30+ REST endpoint хэрэгжүүлсэн.

API Endpoints

  • /api/auth — login, register, OAuth
  • /api/courses — CRUD, enroll
  • /api/tasks — submit, execute
  • /api/ai — AI chat, token balance
  • /api/contest — join, leaderboard
  • /api/payment — QR, history

Validation & Error

  • Zod schema validation бүх input-д
  • 5xx алдааг client-д нуудаг
  • Dual auth: NextAuth + Bearer
  • Rate limiting per IP/user
  • In-memory cache 10s–60s TTL
