Хураангуй

Монгол хэл

ArenaHub нь хиймэл оюун ухаан (AI) болон gamification технологид суурилсан full-stack coding learning platform юм. Хэрэглэгчид програмчлалын хичээл судлах, coding даалгавар бодох, AI assistant-аас тусламж авах, achievement болон leaderboard системээр дамжуулан суралцах идэвхээ нэмэгдүүлэх боломжтой. Систем нь 30 database model, 55 API route, ~350+ coding task бүхий production-level serverless architecture ашиглан бүтээгдсэн бөгөөд 446 translation key бүхий Монгол-Англи localization дэмждэг.

Түлхүүр үг: gamification, AI tutor, coding learning platform, Next.js, PostgreSQL, JWT

English

ArenaHub is a full-stack coding learning platform built using AI and gamification technologies. The platform enables users to study programming lessons, solve coding challenges, receive AI-powered assistance, and increase engagement through achievement and leaderboard systems. The system was developed using modern technologies including Next.js 15, React 19, TypeScript, Prisma ORM, PostgreSQL, and Groq AI with a production-level serverless architecture implementing 30 database models, 55 API routes, ~350+ coding tasks, and 446 translation keys.

Keywords: gamification, AI tutor, coding learning platform, Next.js, PostgreSQL, JWT

6. Системийн Хэрэгжилт

6.1 Frontend Хөгжүүлэлт

Frontend хэсгийг Next.js 15 App Router architecture ашиглан боловсруулсан. Pixel-art dark cyberpunk aesthetic бүхий UI дизайн, responsive layout хэрэгжүүлсэн. State management-д React Context API ашигласан: AuthContext нь хэрэглэгчийн session удирдана, LanguageContext нь МН/EN хэлний солилтыг удирдана.

6.2 Backend Хөгжүүлэлт

Backend нь Next.js API Routes дээр суурилсан serverless architecture ашиглан нийт 55 REST endpoint хэрэгжүүлсэн. Бүх request дээр Zod schema validation ашигласан. 5xx алдааг client-д нуух замаар мэдээлэл алдагдахаас хамгаалдаг.

6.3 Authentication Систем

Систем нь NextAuth.js v4 ашиглан Google болон GitHub OAuth нэвтрэлт хэрэгжүүлсэн. OAuth session-ийг custom JWT токен (15 минутын access + 7 хоногийн refresh) руу хөрвүүлж Bearer auth ашиглан API хамгаалдаг.
